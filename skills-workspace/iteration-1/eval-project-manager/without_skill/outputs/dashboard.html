💡 Khuyến cáo Tuần tiếp theo (20/4 - 24/4)
🔴 P0: Follow-up GotIt OAuth2
Gọi, email, hoặc escalate đến GotIt management. Đây là blocker cho production. Nếu không có response trong 24h, lên kế hoạch alternative (API key, password, webhook flow khác).
🔴 P0: Chốt UI Designs
Bắt đầu ngay hôm nay: Web Admin UI, Miniapp UI, CRM UI cần design mockup và chốt spec. Ước tính công sức. Đây là critical path cho go-live.
🔴 P0: Chốt Mode Ký quỹ với PO/BA
Họp ngay với PO/BA/Finance để confirm: 3 modes hay customize? Nếu cần change, dev cần biết sớm để không phải rebuild.
🟡 P1: Xử lý ZNS hoặc Accept Limitation
Nếu ZNS là MVP requirement, fix ngay. Nếu không, accept limitation và document. Email/SMS là fallback đủ dùng.
🟡 P1: Lên kế hoạch Luồng Đối soát
Khi nào cần delivery? Ước tính chi phí. Có thể là sprint tiếp sau go-live.
🟡 P1: Chuẩn bị Test Script
Lên danh sách test cases cho QA: happy path, edge cases (double-click, timeout, retry), integration (workers + API).
